# Line 對話紀錄備份瀏覽與分析器

這是一個 100% 本地運行、著重隱私的 Line 對話紀錄備份、解析與統計分析工具。不需要安裝任何後端服務，您只需用瀏覽器開啟 `index.html`，即可拖入對話檔案開始使用！

## 🌟 功能特色

- **🔒 隱私安全**：所有解析與照片處理皆在您的個人瀏覽器中完成，不會上傳任何內容至伺服器。
- **💬 精美 Line 聊天介面**：比照 Line App 的經典風格呈現對話氣泡、貼圖與系統訊息，並支援**深色/淺色主題模式**切換。
- **📁 圖片關聯與相簿**：只需將 Line 匯出的照片資料夾拖入瀏覽器中，程式將自動比對訊息時間並在對話中完美展示圖片，同時提供集中式相簿庫。
- **🔍 關鍵字搜尋與篩選**：支援全文檢索、依發言人過濾、依日期範圍篩選對話紀錄。
- **📊 數據可視化統計**：基於 Chart.js 生成：
  - 聊天訊息總數、總天數與圖片計數
  - 聊天成員發言比例圓餅圖
  - 每日/每月聊天熱度折線圖
  - 24 小時活躍時段長條圖
- **💾 本地持續化儲存**：使用瀏覽器內置的 IndexedDB，關閉網頁後再次開啟，先前的匯入紀錄與照片依然存在（可隨時手動刪除）。

## 🚀 快速開始

### 方法 1：直接開啟網頁
1. 下載或複製此專案的檔案：`index.html`、`style.css`、`app.js`。
2. 連按兩下在瀏覽器中開啟 [index.html](file:///d:/GitHub/Line_backup/index.html)。
3. 拖入您的 Line 對話文字檔（`.txt`）或選取匯入。

### 方法 2：使用本地開發伺服器（推薦）
如果您希望有更好的載入性能與檔案存取體驗，可以啟動一個簡單的伺服器。
- **Python**: 在目錄下運行 `python -m http.server 8000`，並在瀏覽器輸入 `http://localhost:8000`。
- **NodeJS (HTTP-Server)**: 執行 `npx http-server`。

## 📝 如何從 Line 匯出資料

### 1. 匯出對話紀錄文字檔 (.txt)
- **iOS / Android**: 進入您想備份的聊天室 -> 點擊右上角選單 -> **「設定」** -> **「傳送聊天紀錄」**。
- 將匯出的 `.txt` 檔案直接拖入網頁的「匯入對話紀錄」區塊中。

### 2. 匯出聊天相片與媒體
- **手機端**：需自行將聊天室中的圖片儲存至手機相簿後匯出。
- **電腦端 (Line Desktop)**：進入聊天視窗 -> 點擊右上角「照片/影片」-> 全選 -> 點擊**「下載」**。下載後的資料夾內即包含所有聊天相片。
- 將該相片資料夾直接拖入網頁的「拖放相片資料夾或檔案」區塊。系統會根據檔名中的時間戳記（例如 `20231024_153012.jpg`）與對話時間做對齊，使 `[圖片]` 轉換為對話中的實體照片。

## 🗄️ SQLite 資料庫匯出工具 (`scripts/sqlite_parser.py`)

如果您取得了已解密的 Line SQLite 資料庫（例如 Root 過的 Android 裝置中的 `talk.db`），我們提供了一個 Python 腳本，可自動將其所有的聊天群組及訊息導出為本 Viewer 支援的 JSON 格式。

### 使用步驟
1. 確保您的電腦上已安裝 Python 3。
2. 執行以下指令：
   ```bash
   python scripts/sqlite_parser.py <資料庫路徑/talk.db> [輸出目錄路徑]
   ```
3. 腳本會在輸出目錄中為每個聊天室生成對應的 `.json` 檔案（例如 `line_backup_親愛的.json`）。
4. 將該 `.json` 檔案拖放到 Viewer 網頁中，即可瞬間載入該群組的完整聊天歷史！
